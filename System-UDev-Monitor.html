Creation

data SourceId

Event source identifier.

udevId :: SourceId

Events are sent out after udev has finished its event processing, all rules have been processed, and needed device nodes are created.

kernelId :: SourceId

Events are sent out just after kernel processes them.

Applications should usually not connect directly to the kernel events, because the devices might not be useable at that time, before udev has configured them, and created device nodes. Use kernelId instead.
